# 샘플 데이터

이 문서는 로컬 시연과 Swagger 테스트에 사용할 수 있는 샘플 문제와 학생 답안을 정리합니다.

문제 생성 API:

```http
POST /api/questions
```

학생 답안 채점 API:

```http
POST /api/grading-requests
```

## 1. 과학: 탄소 중립

### 문제 생성 JSON

```json
{
  "title": "탄소 중립의 의미 설명",
  "subject": "science",
  "content": "탄소 중립이 무엇인지 설명하고, 실천 방법을 두 가지 이상 서술하시오.",
  "modelAnswer": "탄소 중립은 배출한 이산화탄소의 양만큼 흡수하거나 감축하여 실질적인 탄소 배출량을 0으로 만드는 것이다. 실천 방법으로는 대중교통 이용, 에너지 절약, 재생에너지 사용, 나무 심기 등이 있다.",
  "totalScore": 100,
  "rubricItems": [
    {
      "name": "개념 이해",
      "criteria": "탄소 중립의 의미를 정확히 설명한다.",
      "maxScore": 40,
      "sortOrder": 1
    },
    {
      "name": "실천 방안",
      "criteria": "탄소 중립을 위한 실천 방법을 두 가지 이상 구체적으로 제시한다.",
      "maxScore": 40,
      "sortOrder": 2
    },
    {
      "name": "표현 명확성",
      "criteria": "답안이 명확하고 논리적인 문장으로 구성되어 있다.",
      "maxScore": 20,
      "sortOrder": 3
    }
  ]
}
```

### 학생 답안 예시

좋은 답안:

```json
{
  "questionId": 1,
  "studentAnswer": "탄소 중립은 이산화탄소를 배출하지 않는다는 뜻이 아니라, 배출한 양만큼 줄이거나 흡수해서 실질적인 배출량을 0으로 만드는 것입니다. 실천 방법으로는 대중교통을 이용해 자동차 배출가스를 줄이고, 전기를 아껴 발전 과정에서 나오는 탄소를 줄이는 것이 있습니다. 나무 심기나 재생에너지 사용도 도움이 됩니다."
}
```

보통 답안:

```json
{
  "questionId": 1,
  "studentAnswer": "탄소 중립은 탄소를 줄이는 것입니다. 실천 방법은 대중교통을 이용하고 전기를 아끼는 것입니다."
}
```

낮은 품질의 답안:

```json
{
  "questionId": 1,
  "studentAnswer": "탄소 중립은 이산화탄소를 아예 배출하지 않는 것입니다. 그냥 쓰레기를 버리지 않으면 됩니다."
}
```

```json
  "totalScore": 100,
  "rubricItems": [
    {
      "name": "원리 제시",
      "criteria": "민주주의의 기본 원리를 두 가지 이상 정확히 제시한다.",
      "maxScore": 35,
      "sortOrder": 1
    },
    {
      "name": "원리 설명",
      "criteria": "제시한 원리의 의미를 구체적으로 설명한다.",
      "maxScore": 40,
      "sortOrder": 2
    },
    {
```

```json
{

```

보통 답안:

```json
{
  "questionId": 2,
  "studentAnswer": "민주주의에는 국민 주권과 법치주의가 있습니다. 국민이 주인이며 법에 따라 나라를 운영하는 것입니다. 이것은 사회를 공정하게 만들기 때문에 중요합니다."
```


```json

  "content": "다음 글의 중심 내용을 요약하고, 글쓴이가 제시한 근거를 두 가지 이상 찾아 서술하시오. 글: 최근 도시에서는 빗물을 모아 다시 사용하는 시설이 늘고 있다. 빗물 저장 시설은 집중호우 때 물이 한꺼번에 하수도로 흘러가는 것을 줄이고, 저장된 물을 조경용수나 청소용수로 활용할 수 있게 한다. 또한 물 부족 문제를 완화하고 도시의 물 순환을 회복하는 데 도움을 준다.",

      "maxScore": 40,
      "sortOrder": 1
    },
    {
      "name": "근거 파악",
      "criteria": "글쓴이가 제시한 근거를 두 가지 이상 정확히 찾는다.",
```

좋은 답안:

```json
{
  "questionId": 3,
```

```json
{
  "questionId": 3,
  "studentAnswer": "이 글은 빗물을 모아서 쓰는 시설이 좋다는 내용입니다. 빗물을 청소할 때 쓸 수 있고 물 부족 문제에도 도움이 됩니다."
}
```

낮은 품질의 답안:

```json
{
  "questionId": 3,
  "studentAnswer": "이 글은 비가 많이 오면 위험하다는 내용입니다. 물을 아껴 써야 합니다."

```

## 사용 순서

1. Swagger에서 문제 생성 JSON을 `POST /api/questions`로 등록합니다.
2. 응답으로 받은 `id`를 학생 답안 JSON의 `questionId`에 넣습니다.
3. 학생 답안 JSON을 `POST /api/grading-requests`로 제출합니다.
4. 응답의 `gradingResultId`로 `GET /api/grading-results/{gradingResultId}`를 조회합니다.
